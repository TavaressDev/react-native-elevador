# React Native Elevador

Aplicacao mobile desenvolvida com React Native e Expo para simular um painel de elevador com tela de boas-vindas, lista de noticias e alternancia entre modo claro e modo escuro.

## Objetivo

Este projeto foi desenvolvido como atividade da cadeira de Desenvolvimento Mobile da faculdade.

## Funcionalidades

- Tela de boas-vindas
- Navegacao com renderizacao condicional sem uso de router
- Lista de noticias em cards
- Alternancia entre modo claro e modo escuro
- Loader com `ActivityIndicator`

## Tecnologias

- React Native
- Expo
- TypeScript

## Como executar

Instale as dependencias:

```bash
npm install
```

Inicie o projeto:

```bash
npm start
```

Para abrir em uma plataforma especifica:

```bash
npm run android
npm run ios
npm run web
```
